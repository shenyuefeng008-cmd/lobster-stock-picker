# Task Summary 2026-05-21 模拟交易情绪仓位控制

## Objective
为模拟交易引擎添加情绪仓位控制，盘面恶劣时自动减仓/清仓。

## Key Changes

### simulated_trading.py 新增功能
1. **EMOTION_POSITION_RULES** — 涨跌家数→仓位上限映射表
   - <1500冰点：1.0主导，总仓5成，不开新仓（剩余不足5%）
   - 1500-2000冰点修复：1.0主导+3.0熔断，总仓5成
   - 2000-2500修复期：1.0+3.0，总仓9成
   - 2500-3500平稳：2.0+1.0，总仓7成
   - >3500极度高潮：辅助模式，总仓2成

2. **check_position_limit()** — 买入前检查仓位上限，剩余额度不足5%拒绝开仓

3. **emotion_force_sell()** — 情绪极端时强制减仓
   - 极度高潮>3500：清仓所有1.0/2.0持仓（只留3.0趋势）
   - 冰点<1500：减仓到5成以内（亏损股优先卖）

4. **buy()** 新增 up_count 参数，自动按情绪规则计算仓位

### Cron任务更新
- CRON_BUYPOINT_TASK.md：买点触发后调用buy(up_count=...)
- CRON_SELLPOINT_TASK.md：卖点监控+情绪强制减仓
