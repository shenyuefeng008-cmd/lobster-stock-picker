# 龙虾盘中巡检 — 统一买卖点监控（v1 合并任务）

> **频率**：每30分钟（10:00-14:30全覆盖）
> **cron**：`0 10,11,13,14 * * 1-5` + `30 10,12,13,14 * * 1-5` + `50 11 * * 1-5`（跳过11:30，11:50轮次替代原12:00轮次，与午间博客错开）
> **超时**：120秒
> **核心**：一次采集 → **候选池自动补生** → 先卖后买 → 统一输出
> **盘中补生**：若候选池缺失/过期，脚本自动从今日涨停池+板块资金流挖掘回踩标的（无需手动干预）
> **替代**：CRON_BUYPOINT_TASK.md + CRON_SELLPOINT_TASK.md + CRON_LATE_SESSION_TASK.md

## 步骤1.5：盘中动态仓位调节（v1.1新增）

> 每轮巡检自动执行，内置于巡检脚本。无需额外操作。
>
> **逻辑**：每轮巡检重新判定情绪区间 → 计算目标仓位上限 → 对比实际仓位：
> - 实际 > 上限+5%缓冲 → 自动减仓（优先减浮盈少的标的）
> - 实际 < 上限-5%缓冲 → 允许加仓（正常走买点检测）
> - 缓冲区内 → 维持不变（防边缘跳动）

### 1.6.2 搜索新闻（使用 tencent-news 技能）
