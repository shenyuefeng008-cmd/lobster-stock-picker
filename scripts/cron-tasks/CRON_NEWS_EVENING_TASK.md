# 新闻采集任务 — 晚间要闻

> **版本**：v1.0
> **创建日期**：2026-05-31
> **执行频率**：每日 20:00（含非交易日，每天跑）
> **超时**：180s
> **触发方式**：cron → 每日20:00自动执行

## 任务目标

采集收盘后到晚间（15:00-20:00）的重要财经新闻/政策/公告，作为次日盘前的信息储备。
补全 15:05收盘复盘 → 次日07:00盘前 之间的新闻空白。

## 与收盘复盘的区别

- **收盘复盘(15:05)**：写复盘分析 + 采集收盘即时的要闻
- **晚间要闻(20:00)**：采集收盘后数小时的**晚间政策/公告/外盘动态**，为次日盘前提供信息储备

## 采集范围

搜索关键词优先级：
1. `A股 晚间 公告 政策 重要` — 收盘后重要公告/政策
2. `财经 要闻 今日 晚间` — 财经要闻汇总
3. `证监会 IPO 监管 处罚` — 监管动态
4. `美股 欧股 期货 夜盘` — 外盘动态（影响次日A股开盘）
5. 活跃赛道名 + `公告 利好 利空`

### Step 3：筛选与去重

- **去重**：与步骤1提取的已有标题对比，剔除重复条目
- 保留 🔴高/🟡中 级别
- 特别关注：证监会政策、重要公司公告、外盘大跌/大涨（影响次日情绪）
- 🟢低 级别不写入

### Step 4：写入新闻存档

写入 `trading/news/YYYY-MM-DD.md` 的【收盘要闻】区（与15:05收盘复盘写同一区）：

| 发布日期 | 标题 | 来源 | 赛道/板块 | 影响等级 | 核实状态 |
|----------|------|------|-----------|----------|----------|

- 如果文件不存在，先创建（含标题和四个区）
- 如果文件已存在，追加到【收盘要闻】区末尾
- 如有与催化日历相关新闻，同时写入【催化剂相关新闻】区

### Step 5：输出报告

```
📰 晚间要闻 YYYY-MM-DD（20:00）
- 采集新闻：X条
- 去重跳过：Y条
- 新增条目：
  1. [标题] → [赛道] [影响等级]
  2. ...
```

如无新要闻：
```
📰 晚间要闻 YYYY-MM-DD（20:00）
- 采集新闻：0条
- 无新增要闻
```

## 注意事项

1. 非交易日也要跑（周末政策/外盘动态同样重要）
2. 搜索失败不阻塞，记录错误后输出空报告
3. 不覆盖已有条目，只追加
4. 外盘大涨/大跌（>1%）必须标 🔴高
5. 来源分级规则见 `trading/news/README.md`
