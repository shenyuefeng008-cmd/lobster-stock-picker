## 一、底层投资范式

### 新旧范式对比

| 维度 | 旧范式（2022-2024） | 新范式（2025-2026） |
|------|---------------------|---------------------|
| 核心变量 | 需求侧（订单、用户数、预期空间） | **供给侧（产能锁定、产能壁垒、交付确定性）** |
| 估值锚 | 业绩兑现（EPS） | **产能兑现（Capacity delivery）** |
| 估值时间窗口 | 1-3年 | **10-30年（hot AI summer → 打到30年估值）** |
| 估值方式 | 周期股（存储、零部件） | **PE扩张（外资已将存储从周期改为PE估值）** |
| 市场原谅 | — | 有产能壁垒一直给正反馈，甚至溢价 |
| 市场惩罚 | 需求miss | 产能miss但有壁垒保护 vs 供给无壁垒的miss |

### 三类赛道状态

| 类型 | 定义 | 操作策略 |
|------|------|----------|
| "躺平简单题" | 一定会发生、空间大、能兑现到自身 | 大多数已涨3-10倍，进入高位。**持有，不追** |
| "轮动补涨" | 20个方向最终都会轮动到，需要新变化催化 | 未轮到或轮到后还有空间。**等边际变化信号** |
| "鱼尾/兑现期" | 催化一致性强、一致性高潮 | 陶瓷基板、机器人龙头等。**克制不追** |

## 二、四层嵌套决策体系（L1-L4）

### 框架全景

```
┌─────────────────────────────────────────────────┐
│                  L1 市场环境层                      │
│   定义：现在处于什么市场阶段？量能？主线？            │
│   约束：决定仓位上限与策略类别                      │
│   → 回答"能不能做"                                 │
├─────────────────────────────────────────────────┤
│                  L2 催化剂层                        │
│   定义：什么催化剂在驱动市场？级别多强？              │
│   约束：决定仓位方向与分配比例                      │
│   → 回答"做什么方向"                               │
├─────────────────────────────────────────────────┤
│                  L3 策略执行层                      │
│   定义：当前适用什么策略？如何执行？                 │
│   约束：决定买卖时机、仓位管理、止盈止损              │
│   → 回答"怎么做"                                   │
├─────────────────────────────────────────────────┤
│                  L4 错误反馈层                      │
│   定义：哪些行为模式反复导致亏损？                   │
│   约束：决定自我纠偏与系统进化方向                   │
│   → 回答"怎么进化"                                 │
└─────────────────────────────────────────────────┘
```

### L1 市场环境层 — 仓位上限

| 市场阶段 | 量能特征 | 主线特征 | 仓位上限 |
|----------|----------|----------|----------|
| 主升浪加速期 | 3万亿+ | 3条以上主线并行 | 80-100% |
| 结构性牛市 | 1.5-2.5万亿 | 1-2条主线主导 | 60-80% |
| 分歧调整期 | 缩量至2万亿以下 | 主线回调、资金切换 | 30-50% |
| 缩量震荡期 | 1万亿以下 | 方向不明 | 10-30% |
| 熊市下行期 | 持续缩量 | 无明确主线 | 0-10% |

**L1核心约束**：仓位上限由L1决定，不由个股逻辑决定。"牛市策略被套用至熊市"是最常见的亏损根源。

### L2 催化剂层 — 四级催化剂

| 层级 | 命名 | 定义 | 仓位上限 | 持有周期 |
|------|------|------|----------|----------|
| Tier 1 | 范式级 | 彻底改变行业TAM或竞争格局 | 20-25% | 季度至年度 |
| Tier 2 | 周期确认级 | 验证供需缺口持续时间 | 15-20% | 月度至季度 |
| Tier 3 | 资金驱动级 | 大额订单、政策催化 | 10-15% | 周度至月度 |
| Tier 4 | 补涨扩散级 | 主线关联间接的扩散 | 5-10% | 日度至周度 |

**催化共振**：Tier 1 + Tier 2 + Tier 3 同一天共振 = 市场做多最优环境。

### L3 策略执行层 — 策略与市场阶段匹配

| 市场阶段（L1） | 适用策略 | 核心原则 |
|----------------|----------|----------|
| 主升浪加速期 | 趋势持有为主（3.0） | 持有不动、回调加仓、不频繁换股 |
| 结构性牛市 | 主线集中聚焦（3.0+2.0） | 最强1-2条主线、控制跟风仓位 |
| 分歧调整期 | 调仓切换（2.0） | 分歧日不接飞刀、等企稳再入场 |
| 缩量震荡期 | 低吸轮动/空仓（1.0+2.0） | 快进快出或直接空仓 |
| 熊市下行期 | 空仓/极小仓位 | 不参与、保护本金 |

### L4 错误反馈层 — 常见错误

| 错误类型 | 表现 | 防范措施 |
|----------|------|----------|
| 风格错配 | 牛市做连板、熊市做趋势 | 开仓前检查：当前阶段适用什么策略？ |
| FOMO追高 | 怕踏空、涨停板追入 | Tier 3/4催化时强制冷却期30分钟 |
| 过早止盈 | 趋势持有变成短线套利 | 按催化等级执行最低持有天数（见趋势容量池买卖点规则） |
| 锚定效应 | 被买入成本锚定 | 亏损持仓忽略成本，只问"现在值不值得持有" |
| 确认偏误 | 只找支持证据、忽略风险 | 重仓方向强制写出三条反对理由 |
| 过度交易 | 频繁换股 | 主升浪期实施"禁售期" |

## 四、产业图谱自我进化规则

### 进化触发条件

以下情况发生时，必须主动更新本文件：

| 触发条件 | 进化行动 |
|----------|----------|
| 涨停池出现非11大赛道的新方向（连续2日≥3家涨停） | 标记为"新候选赛道"，收集信息后提议新增 |
| 11大赛道中某赛道的催化节点兑现/落空 | 更新该赛道状态（🔴/🟡/🟢） |
| 赛道状态从🟡变为🔴（或反向） | 更新轮动总表排名 |
| 出现Tier 1级别催化剂（范式级） | 更新L2催化剂层，可能触发L1环境变化 |
| 同一赛道连续3日涨停集中 | 更新该赛道为"正在轮动"状态 |
| 某赛道龙头跌停/板块集体跳水 | 标记为"风险释放"或"逻辑证伪" |
| 🟢 赛道连续 4 周无涨停验证 | 标记为"候选淘汰"，降级至「待观察区」|
| 🟢/🟡 赛道核心催化全部落空且无新催化 | 标记为"逻辑证伪"，进入淘汰流程 |
| 🟢 赛道连续 8 周无任何验证 + 无池中标的 | 从轮动总表正式移除，归档至「历史赛道」 |
| 框架总赛道数超过 22 | 强制淘汰排名最末的🟢赛道，保持总数≤22 |

### 每日进化流程（嵌入收盘复盘）

1. 对比今日涨停板块分布 vs 11大赛道 → 新赛道信号？
2. 检查催化日历 → 近期有无兑现/落空？
3. 更新赛道状态标签（🔴/🟡/🟢）
4. 如有重大变化 → 更新本文件 + 写入进化日记

### 每周深度进化（周日20:00）

1. 搜索龙虾短线知识库最新"学霸圈深度整理"
2. 提取新的边际变化信息
3. 更新11大赛道的催化节点和状态
4. 检查是否需要新增/删除赛道
   a) 遍历所有🟢赛道，检查距离最后一次涨停验证的天数
   b) ≥28 天无验证 → 标记"候选淘汰"，移动到文件末尾「待观察区」
   c) ≥56 天无验证 + 趋势容量池中无此赛道标的 → 正式移除，写入「历史赛道归档」
   d) 新增赛道时，如框架总数超过22，按"最近无验证天数"和"确定性评级"加权排序，淘汰末尾赛道
5. 更新轮动节奏预测

趋势容量池（trading/趋势容量池.md）从11大赛道中筛选标的，标准：

| 标准 | 说明 |
|------|------|
| 赛道状态 | 必须为🔴或🟡（L1/L2），L3/L4仅观察不入池 |
| 产业逻辑 | 有明确的Tier 1或Tier 2催化剂支撑 |
| 容量 | 日成交额≥5亿（大票）或≥2亿（中票且赛道确定性★★★★+） |
| 均线 | 5日>10日>20日（多头排列），或5日刚上穿10日（金叉确认） |
| 位置 | 非一致性高潮后的高位（排除🔴"一致性高潮边缘"的龙头） |
| 板块地位 | 所在细分赛道的前2-3名核心标的 |
