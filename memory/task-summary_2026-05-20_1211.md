## 四、已固化规则汇总（v2.4最终版）

### 1. 情绪切换规则
```
<1500      → 1.0主导，总仓≤5成
1500-2500  → 1.0+3.0，总仓≤9成
2500-3500  → 2.0+1.0，总仓≤7成
>3500      → 辅助模式，总仓≤2成

模糊区定义：阈值±50（如1450-1550）→ 保守处理
```

### 2. 冰点修复确认（v2.4新）
```
🔴冰点      → 涨跌家数<1500，1.0主导，3.0休眠
🧊冰点持续  → 涨跌家数<2000，3.0强制休眠
📉修复确认  → 连续2日>1700（从1500上调）
✅激活      → 连续2日>2500，3.0正式激活
```

### 3. 分维度止损（v2.3）
```
1.0：硬止损 -5%，时间止损 第3日未涨停
2.0：硬止损 -7%，板块止损 板块跌>3%+个股破MA5
3.0：技术止损 MA5下穿MA10，逻辑止损 催化落空
```

### 4. 持有期硬约束（v2.4新）
```
Tier-1（核心赛道龙头）：≥5日
Tier-2（延伸赛道前排）：≥3日
提前退出条件：仅逻辑证伪（政策逆转/技术颠覆）
```

### 5. 出池规则（v2.2对称）
```
预警：连续2日<4项满足 / 赛道🔴→🟢 / 板块地位跌出前10
确认：连续5日<4项满足
强制：MA5<MA10且3日不收回 / 连续2日<2000 / 逻辑破坏
观察区：30日冷静期，不得重新入池
```

### 6. 卖点量能分化（v2.4新）
```
放量下穿（>1.5倍均量）：1日确认
缩量下穿：3日确认
大幅下穿（>3%）：2日确认
```

### 7. 板块确认时间（v2.4新）
```
上午确认：11:30前 ≥2家涨停封死 + 板块涨幅>2%
全天确认：≥3家涨停 且 封板>30分钟
排除：14:50后偷鸡涨停不计入
```
