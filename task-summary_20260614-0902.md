# 云同步Cron任务创建 | 2026-06-14 09:02

## 目标
用户要求将龙虾系统每日收集的新闻、催化日历、产业图谱等数据自动同步到腾讯SMH云存储（乐享），每日02:00执行。

## 实现方案
1. **同步脚本**：`scripts/lobster_cloud_sync.sh`
   - 核心数据10个文件（催化日历/产业图谱/趋势容量池/关注股/交易追踪/模拟持仓/催化剂数据库/config/rules）→ 覆盖式上传，始终最新版
   - 每日归档2个文件（新闻/日志）→ 按日期命名追加
   - 直接用curl调用 `localhost:19000/proxy/qclaw-cos/upload` API

2. **Cron任务**：`龙虾数据云同步`（id: ae6053d3）
   - 每日02:00执行
   - 读取 CRON_CLOUD_SYNC_TASK.md 指令
   - 推送通道：yuanbao

3. **SMH目录结构**：
   - 一级目录 `1gwpiwf3hr163jz5/`（已存在）
   - 文件命名：`lobster-最新-xxx.md`（核心）/ `lobster-YYYY-MM-DD-xxx.md`（归档）
   - SMH不支持自动创建二级子目录，因此用文件名前缀代替子目录组织

## 测试结果
- 首次同步 2026-06-14：11/12 文件成功（1个memory日志当天不存在，属正常）
- 云端路径：`1gwpiwf3hr163jz5/lobster-最新-催化日历.md` 等

## 遗留
- SMH文件保留30天自动清理，长期归档需考虑定期重新上传或换用其他存储
