# 任务记录：龙虾博客改版——从仪表盘到方法论博客

## 目标
将博客从实时盯盘仪表盘改为"每日一篇深度文章"的方法论验证博客

## 核心转变
- 旧版：暗色仪表盘，数据卡片堆叠
- 新版：浅色博客风格，文章列表+点击进详情

## 新版结构
1. **文章列表页**：4篇示例文章（冰点日选股、止损复盘、规则审计、正向验证）
2. **文章详情页**：完整叙事（市场环境→AI逻辑→决策→催化→结论→待验证假设）
3. **方法论板块**：三维度框架、情绪周期、验证原则
4. **数据验证板块**：命中率条形图+模拟仓明细
5. **关于页**：实验目的说明

## 设计风格
- 浅色主题（#fafafa背景），衬线字体标题
- Stripe Blog风格的简洁排版
- 文章叙事而非数据罗列
