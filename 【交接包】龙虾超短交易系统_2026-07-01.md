## 📋 系统概述

这是一套**全自动A股超短线交易系统**，模拟100万初始资金，通过20个定时任务（cron）实现：

- **盘前选股** → 生成候选池（1.0一进二 / 2.0板块卡位 / 3.0趋势低吸）
- **竞价买入** → 09:30自动执行
- **盘中巡检** → 10:00-14:30每30分钟检测买卖点
- **收盘复盘** → 15:05自动归档 + IMA同步
- **每日进化** → 00:10分析feedback → 自动调整参数

**核心逻辑**: 三维度框架（点·线·面）+ 情绪矩阵 + 催化剂评分

## 🔧 核心模块说明

### 1. 模拟交易核心 (`simulated_trading.py`)
- **功能**: 买入/卖出/持仓管理/资金计算
- **关键函数**:
  - `buy(code, name, price, reason, dimension)` — 买入
  - `sell(code, price, reason, sell_type)` — 卖出
  - `status()` — 输出持仓状态
  - `validate_capital()` — 资金一致性校验
- **数据文件**: `trading/模拟持仓.json`
- **⚠️ 重要**: 每次交易后自动保存 `模拟持仓.json`，多个脚本会读取此文件

### 2. 盘中巡检 (`lobster_intraday_patrol.py`)
- **功能**: 检测买卖点 + 执行交易
- **买点检测**: `detect_buypoints()` — 回踩均线/突破前高
- **卖点检测**: `detect_sellpoints()` — 硬止损/时间止损/分时止盈/情绪强制卖出
- **盘中候选池补生**: `check_and_regenerate_candidates()` — 候选池缺失时自动从涨停池挖掘
- **⚠️ 重要**: 此脚本被cron调用，输出到 `delivery`

### 3. 盘前选股引擎 (`lobster_premarket_engine.py`)
- **功能**: 07:00生成候选池
- **输出**: `/tmp/lobster_premarket_candidates.json`
- **候选池结构**:
  ```json
  {
    "date": "2026-07-01",
    "candidates": {
      "1.0一进二": [{"代码":"300671", "名称":"富满微", ...}],
      "2.0板块卡位": [...],
      "3.0趋势低吸": [...]
    }
  }
  ```

### 4. 情绪获取 (`get_market_sentiment.py` v6)
- **主源**: 新浪财经（采样500只，均匀覆盖）
- **降级**: 腾讯采样（覆盖不均，已弃用为主源）
- **输出**: `{"up": 4576, "down": 869, "zt": 19, "dt": 1}`
- **缓存**: `/tmp/lobster_sentiment_cache.json`（30分钟TTL）
- **⚠️ 重要**: 情绪数据直接影响买卖决策，必须准确
