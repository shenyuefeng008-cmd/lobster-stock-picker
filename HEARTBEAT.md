# 龙虾超短交易心跳系统 v2.1（精简版）

> 完整规则见 `trading/heartbeat-rules-full.md`

## 时段与P0任务

交易日盘前/早盘/午间/尾盘/收盘 → 执行P0。非交易时段 → 降频。

**P0前置**：`memory/交易日判断.md` 确认是否交易日。

## P0 情绪面板（每轮心跳必做）

1. 涨跌家数（legulegu.com）→ 情绪周期 + 维度判定
2. 指数涨跌（qt.gtimg.cn）→ 上证/深证/创业板
3. 涨停数/炸板率（akshare）

**维度判定**：
- <1500 → 1.0主导，仓位上限5成
- 1500-2000 → 1.0主导，仓位上限5成（3.0熔断）
- 2000-2500 → 1.0+3.0，总仓位上限9成（1.0每仓3成，3.0按矩阵5-25%）
- 2500-3500 → 2.0+1.0，5-7成
- >3500 → 辅助模式，1-2成

## P1 三维度买点（情绪可操作时）

- 1.0：一进二竞价 + 分歧低吸
- 2.0：板块爆发检测 + 高低切
- 3.0：趋势池回踩均线低吸（不追涨停）

详细信号条件 → 读 `trading/heartbeat-rules-full.md`

## P2 低频率

- 板块轮动扫描（每2h）
- 产业图谱快查（每日一次）

## 紧急告警

| 条件 | 动作 |
|------|------|
| 涨跌家数<1500 | 🚨 冰点，主导→1.0 |
| 涨跌家数>3500 | 🔴 极度高潮，切换辅助模式 |
| 炸板率>35% | ⚠️ 风险，封板意愿弱 |

详细告警条件 → 读 `trading/heartbeat-rules-full.md`
